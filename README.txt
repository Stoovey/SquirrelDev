This is a development build.

This is a 4 player game. The aim of the game is to collect the acorn and place it into the den that spawns around the map. The game supports four Xbox 360 pads.


Play with controller.
Left analog stick	=	Movement.
Right analog stick	=	Aim
Right Trigger		=	Throw Acorn while holding
B			=	Tackle someone while they are holding the acorn
A			=	Jump
